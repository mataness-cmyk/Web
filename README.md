# Web
“A front-end web application for a digital library, allowing users to browse and access books from anywhere. Designed to be simple, user-friendly, and accessible for everyone.”

## Features
- Browse and view books online
- Search books by title, author, or category
- Responsive design for mobile, tablet, and desktop
## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open the `index.html` file in a web browser
2. Explore the library interface
3. Use the search bar to find specific books
